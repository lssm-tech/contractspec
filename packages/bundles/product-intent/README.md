# @contractspec/bundle.product-intent

Website: https://contractspec.io/

Bundle that wires the product-intent module to an AI model runner and a deterministic evidence retriever.

## What it provides

- AI-backed model runner for product-intent prompts.
- A ready-to-use `createProductIntentService` that combines evidence retrieval with orchestration.

## Installation

```bash
bun add @contractspec/bundle.product-intent
```
