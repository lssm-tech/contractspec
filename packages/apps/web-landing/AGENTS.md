# AI Agent Guide ‚Äî `@contractspec/app.web-landing`

Scope: `packages/apps/web-landing/*`

Marketing landing page and documentation site for ContractSpec. Next.js application consuming `bundle.marketing` and `bundle.library`.

## LLM/Agent entry points

- **`/llms`** (aliases: `/llms.txt`, `/llms.md`) ‚Äî Global monorepo summary; static file at `public/llms.txt`
- **`/docs/llms.txt`** ‚Äî Same content; docs subpath for Agent-Friendly Documentation Spec discovery
- **`/llms-full.txt`** (alias: `/llms-full`) ‚Äî Full content: all package READMEs aggregated; generated by `bun run llms:generate`. Large; prefer `/llms/[slug]` for agent fetches.
- **`/llms/[slug]`** ‚Äî Per-package guide. Slug = package name without @contractspec (e.g. `lib.ai-agent`, `bundle.library`). Returns README as text/plain.
- **Subdomain**: `llms.contractspec.io` ‚Üí serves `/llms.txt`
- **llms-txt-directive**: Docs pages include blockquote "For AI agents: Documentation index at /llms.txt"
- **robots.txt**: Explicit allow for GPTBot, ClaudeBot, PerplexityBot, etc.
- **sitemap**: Includes /llms, /llms.txt, /llms-full.txt, /docs/llms.txt
