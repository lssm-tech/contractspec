# web-landing

Marketing landing page and documentation site for ContractSpec.

## Quick Context

- **Type**: Next.js application
- **Purpose**: Marketing site and docs
- **Bundles**: `bundle.marketing`, `bundle.library`

## Key Directories

- `src/app/(landing-marketing)/` — Marketing pages
- `src/app/(docs)/` — Documentation pages

## Commands

```bash
bun dev    # Development server
bun build  # Production build
bun start  # Production server
```

## Features

- Marketing landing page
- Documentation pages
- Pricing and contact
- Template gallery
- Newsletter signup

## Styling

Uses Tailwind CSS v4 with PostCSS.
