# `@contractspec/lib.notification`

Library-first notification helpers for ContractSpec applications.

This package owns reusable notification entities, schema contributions, channels, templates, and i18n catalogs. Canonical notification contracts, capability metadata, and feature metadata are re-exported from `@contractspec/lib.contracts-spec/notifications`.

## Compatibility

- `notificationsSchemaContribution.moduleId` is `@contractspec/lib.notification`.
- `legacyNotificationsSchemaContribution.moduleId` is `@contractspec/module.notifications`.
- The deprecated `@contractspec/module.notifications` package remains available as a compatibility shim for existing consumers.
