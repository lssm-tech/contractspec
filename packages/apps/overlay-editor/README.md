# ContractSpec Overlay Editor

This Next.js app previews and signs OverlaySpecs. It ships with:

- Visual field toggles (hide/show, rename, reorder).
- Live JSON preview backed by `@lssm/lib.overlay-engine`.
- Server action for Ed25519 signing using PEM-encoded keys.

## Development

```bash
bun install
cd packages/contractspec/packages/apps/overlay-editor
bun dev
```
