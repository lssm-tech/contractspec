# ContractSpec JetBrains Plugin Changelog

All notable changes to the ContractSpec JetBrains Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-01-XX

### Added
- Initial release of ContractSpec JetBrains Plugin
- Complete feature parity with VS Code extension
- Real-time spec validation with inspections
- Specs Explorer tool window with grouping modes
- Dependencies visualization
- Build Results tracking
- Features and Integrity Analysis views
- LLM Tools for AI-assisted development
- Node.js bridge for TypeScript service integration
- Live templates for common spec patterns
- OpenAPI import/export functionality
- Registry browsing and example initialization
- MCP integration for documentation search
- Setup wizard and health check (Doctor)
- Telemetry with privacy controls
- Comprehensive settings configuration

### Technical
- Kotlin-based implementation for JetBrains Platform
- JSON-RPC communication with Node.js bridge server
- Gradle build system with IntelliJ Platform Plugin
- Support for WebStorm and IntelliJ IDEA Ultimate
- Compatible with IDE versions 2024.3+
