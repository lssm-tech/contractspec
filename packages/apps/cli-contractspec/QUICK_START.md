# ContractSpec CLI Quickstart

Use the CLI-first onboarding flow to generate repo-local guidance before you start authoring new specs.

```bash
contractspec quickstart
contractspec onboard
contractspec init
contractspec validate
```

## Track-Focused Onboarding

```bash
contractspec onboard contracts
contractspec onboard ui-design
contractspec onboard knowledge
contractspec onboard ai-agents
contractspec onboard learning-journey
```

Use `--example <key>` to initialize a starter example stub under `.contractspec/examples/`.
