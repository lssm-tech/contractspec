Add a transparent adaptive-experience foundation for ContractSpec personalization.

`@contractspec/lib.personalization` now separates interaction preferences from behavior support strategy, exposes behavior signals as evidence instead of labels, provides runtime-only adaptive experience resolution, and adds explainable onboarding and evolution helpers for scoped, reversible adaptation suggestions.

The package docs now explain how builders should use these primitives without creating hidden profiles, personality classifiers, authorization bypasses, or manipulative personalization flows.
