Add a first-class harness system for controlled inspection, testing, evaluation, and proof generation.

- add `platform.harness` contracts, capabilities, events, scenario and suite specs, data views, and audit presentation support in `@contractspec/lib.contracts-spec`
- add the mode-agnostic harness runtime in `@contractspec/lib.harness` with policy classification, evidence normalization, assertion evaluation, replay bundles, and scenario/suite runners
- add `@contractspec/integration.harness-runtime` with target resolution, artifact stores, deterministic browser and visual adapter surfaces, sandboxed code execution, and MCP exposure helpers
